# JSONAPI Redux Data


## Frontend 
- install the dependencies using 
  ``` 
  yarn
  ```
- run the FE application using 
  ```
  yarn start 
  ```


## Backend

- Go to todo-jsonapi-server
  ```
  cd ../todo-jsonapi-server
  ```
- Install server dependencies
  ```
  yarn
  ```
- Setup the environment
    *   `lux db:reset`
    *   `lux db:migrate`
    *   `lux db:seed`
    *   `lux serve`
- Run the server using 
  ```
  yarn start
  ```

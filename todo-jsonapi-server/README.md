# JSONAPI Redux Data

## Backend

- Install server dependencies
  ```
  yarn
  ```
- Run the server using 
  ```
  yarn start
  ```

## Frontend 

- Go to examples
  ```
  cd ../example/todo-app
  ```
- install the dependencies using 
  ``` 
  yarn
  ```
- run the FE application using 
  ```
  yarn start 
  ```

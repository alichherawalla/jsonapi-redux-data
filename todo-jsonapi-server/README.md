# JSONAPI Redux Data

## Backend

- Install server dependencies
  ```
  yarn
  ```
- Setup the environment
    *   `lux db:reset`
    *   `lux db:migrate`
    *   `lux db:seed`
    *   `lux serve`
- Run the server using 
  ```
  yarn start
  ```

## Frontend 

- Go to examples
  ```
  cd ../example/todo-app
  ```
- install the dependencies using 
  ``` 
  yarn
  ```
- run the FE application using 
  ```
  yarn start 
  ```
